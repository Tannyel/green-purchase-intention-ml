# Predicting the Green Purchase Intention of Generation Z Café Consumers in Manila using Machine Learning Techniques

> **International Conference Publication | Business Analytics Research**

This research applies supervised machine learning techniques to predict the green purchase intention of Generation Z café consumers in Manila. The study identifies the key factors influencing environmentally conscious purchasing behavior and demonstrates how machine learning can support data-driven decision-making in sustainable business strategies.

---

# Business Problem

Although environmental awareness among Generation Z continues to grow, businesses still struggle to identify which factors truly influence consumers to choose environmentally sustainable cafés. Traditional statistical approaches often fail to capture complex relationships between consumer behavior and purchasing decisions.

---

# Research Objective

This study aims to identify the most significant predictors of green purchase intention among Generation Z café consumers in Manila by comparing the predictive performance of multiple supervised machine learning algorithms.

---

# Dataset

| Category | Details |
|-----------|---------|
| Target Population | Generation Z Café Consumers |
| Location | Manila, Philippines |
| Respondents | **444** |
| Data Collection | Survey Questionnaire |
| Analysis | Supervised Machine Learning |

---

# Machine Learning Models

The study evaluated six supervised machine learning algorithms:

- Decision Tree (DT)
- Random Forest (RF)
- Gradient Boosting (GB)
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)
- XGBoost (XGB)

---

# Key Findings

The research identified several factors that significantly influence green purchase intention among Generation Z consumers.

### Strongest Predictors

- Subjective Norms
- Green Perceived Quality
- Green Self-Identity

### Lower Predictors

- Environmental Awareness
- Eco-Social Benefits

Random Forest and Gradient Boosting achieved the highest predictive performance, demonstrating the effectiveness of ensemble learning methods for consumer behavior prediction.

---

# Model Performance

Highlights include:

- 444 survey respondents
- 6 Machine Learning Models
- Up to **81% prediction accuracy**
- Comparative model evaluation using Accuracy, Precision, Recall, F1-Score, Hamming Score, Kappa Score, and AUC-ROC.
